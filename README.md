# Frontend Pin Drop Map Example

This example demonstrates how to create an interactive map with custom location pins using only frontend technologies (HTML, CSS, JavaScript) and the Leaflet.js library. Location data is hardcoded directly into the JavaScript, eliminating the need for a backend server to store or serve map data.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in any web browser.
